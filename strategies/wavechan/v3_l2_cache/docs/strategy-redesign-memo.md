### _get_large_trend() 算法更新

**旧算法**（已废弃）：
```python
# 数笔的个数比率
ratio = len(down_bis) / len(up_bis)

# 用高低点同向变化判断趋势

# 下跌趋势 = 更低的高点 AND 更低的低点
# 其他 = neutral
```

**依据**：老板教的方法 - 趋势 = 高低点在同一方向变化
