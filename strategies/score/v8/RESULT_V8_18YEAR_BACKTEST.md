## 三、核心发现

### 3.1 策略性格
- **趋势跟随型**：核心收益来自牛市（2014-2017年夏普1.0-1.7）
- **熊市极弱**：2010-2012连续两年负收益，2011-2012最大回撤-35.4%
- **震荡市中收益**：年化+5~13%，胜率20-30%

### 3.2 盈亏比分析
- 止损 4% vs 止盈 20% → 风险收益比 **1:5**
- 盈亏平衡胜率：≥17%（1:5 RR）
- 牛市胜率 35%+ → 正期望；熊市胜率 15-20% → 负期望

### 3.3 熊市交易过载现象
- 负收益窗口平均交易 **110笔**，正收益窗口平均 **80笔**
- 原因：熊市RSI超卖信号增多，频繁触发买入→止损
- 解决方向：仓位管理（非预判市场）

## 四、仓位管理方案（待回测验证）

### Trader 建议：连续止损降仓规则

```
触发条件 → 仓位
默认 → 100%
连续3次止损 → 50%
连续5次止损 → 25%
连续2笔盈利 → 恢复+25%
连续4笔盈利 → 恢复100%
```
