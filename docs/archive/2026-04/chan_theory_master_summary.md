## 一、缠论买卖点完整定义表

### 1.1 三大买点定义（缠中说禅核心）

| 买点 | 缠论名称 | 精确定义 | 波浪理论对应 |
|------|---------|---------|------------|
| **一买** | 第一类买点 | 下跌趋势最后一个中枢下方，出现**趋势背驰**（两中枢趋势后，后段力度弱于前段），标志着整个级别下跌走势的终结 | 约等于 **C浪终点**（熊市最后一跌），或 **W4/W2终点**（牛市回调终点） |
| **二买** | 第二类买点 | 一买后次级别向上走势结束，**再次下跌不创新低**的位置。是一买后的第一次空头回拉不创新低 | 对应 **W2终点** 或 **W4终点**（牛市回调的两个最佳买入点） |
| **三买** | 第三类买点 | 向上离开中枢后，**回试不破中枢上沿（ZG）**的位置。确认趋势将延续 | 对应 **W5起点稍后**（第5浪中途），或更大级别的新推动浪启动后回踩 |

### 1.2 识别条件详解

#### 【一买】识别条件（最严苛，胜率最低但利润最大）

```
必要条件：
1. 存在至少两个同级别中枢（下跌中枢）
2. 离开中枢的走势与进入中枢的走势出现背驰（力度比较）
3. 背驰段内部也出现背驰（区间套确认）

辅助条件：
4. 成交量：背驰段下跌时成交量萎缩（恐慌盘出尽）
5. 分型：底分型形成（但注意中继分型陷阱）
6. MACD：黄白线在0轴以下明显背离
7. 位置：接近重要支撑位（年线/前低/黄金分割位）

【警告】：一买是最难捕捉的买点，条件严苛且容易中继（底分型停顿后继续下跌）
```

#### 【二买】识别条件（最安全，实战最可靠）

```
必要条件：
1. 一买已经形成（前提）
2. 随后的次级别向上走势已经完成（顶分型确认）
3. 再次下跌，但**低点不破一买低点**（关键！）

辅助条件：
4. 回调位置：在W1涨幅的50%-61.8%斐波那契位（标准回撤）
5. 成交量：回调时量能萎缩（主力未出货）
6. 分型：底分型形成
7. 均线验证：回调不破重要均线（如20日/60日均线）

【注意】：二买最多出现三个，超过三个则说明走势弱，可能持续震荡
```

#### 【三买】识别条件（力度最强，确认最快）

```
必要条件：
1. 存在一个向上离开中枢的走势（离开段）
2. 随后的次级别回试，低点**不跌破中枢上沿ZG**
3. 三段次级别回踩完成（标准形式）

辅助条件：
4. 离开段放量：离开中枢时成交量放大
5. 回踩缩量：回踩确认时量能萎缩
6. 均线共振：回踩不破均线

【陷阱】：如果回踩跌破ZG，则变成三卖！必须严格止损
```

## 二、实战胜经提炼

### 2.1 哪些买点实战中最可靠？

根据实战统计和理论研究，**买点可靠性排序**如下：

#### 🥇 第一梯队（胜率 70-90%，强烈推荐）

**1. W2回调二买（牛市最佳买点）**
```
识别要点：
- W2回撤W1涨幅的50%-61.8%（斐波那契标准位）
- 成交量：回调时量能萎缩至W1的40%以下
- 分型：底分型确认
- 止损：设在W1起点下方

【为什么最可靠】：
- W2是牛市确认后的第一次回调，主力建仓充分
- 回撤幅度可控（标准50%-61.8%），止损位明确
- W3通常是5浪中最长的一浪，上涨空间大
- 符合"趋势在休息时买入"的原则
```

**2. W4回调二买（牛市第二次买入机会）**
```
识别要点：
- W4回撤W3涨幅的38.2%-50%（浅回调）
- W4低点不跌破W1高点（铁律！）
- 横盘整理形式优于急剧下跌形式
- 底分型 + 放量阳线确认

【为什么可靠】：
- W4是顺势买入的第二次机会
- 浅回调说明趋势强劲
- W5即使失败也有W3高点作为安全垫
```

**3. C浪终点三买（熊市最佳买点）**
```
识别要点：
- C浪走出5浪结构（内部5个子浪完成）
- C浪长度 = A浪 × 1.618（斐波那契目标达成）
- 成交量：恐慌抛盘后量能萎缩
- 出现底分型 + 大阳线

【为什么可靠】：
- C浪是熊市最后一跌，情绪绝望
- 之后是大级别反转，上涨空间最大
- 量化统计：C浪终点后平均涨幅超过100%
```

#### 🥈 第二梯队（胜率 50-70%，可接受）

**4. 三买（中枢上沿回踩不破）**
```
可靠性条件：
- 日线级别中枢 + 30分钟三段回踩
- 离开中枢时放量，回踩缩量
- 均线（20日）支撑有效

【陷阱规避】：
- 复杂中枢（延伸、扩张）后的三买力度弱
- 前置三买（小级别）容易被骗
- 必须设止损：跌破ZG立即走
```

**5. 大级别一买（周线/月线级别）**
```
可靠性条件：
- 周线底分型 + 趋势背驰
- 多级别共振（周线一买 + 日线二买）
- 大盘环境配合（资金翻多）

【注意】：
- 小级别一买陷阱多（30分钟/60分钟）
- 日线级别以上的一买才具有操作价值
```

#### 🥉 第三梯队（胜率 30-50%，谨慎使用）

**6. 一买（最低点抄底）**
```
问题：
- 条件严苛，经常出现中继底分型
- 需要区间套精确确认（小转大）
- 实战中容易被套

【实战原则】：
- 小级别一买直接放弃
- 只操作日线及以上级别
- 必须严格止损（亏损超过5%立即走）
```

**7. B浪反弹（绝对不追！）**
```
【致命陷阱】：
- B浪是熊市中最大的陷阱
- 表面看是"抄底机会"，实际是刀口舔血
- 统计数据：B浪追高亏损概率超过80%

【原因分析】：
- B浪反弹通常无量（主力未参与）
- 随后C浪跌幅往往是A浪的1.618倍
- 2015年股灾：5178点B浪反弹追高者损失惨重
```

### 2.2 实战中最常见的陷阱

| 陷阱类型 | 具体表现 | 识别方法 | 应对策略 |

#### 问题四：止损位设置可以更精细

**修正建议**：
```python
# 更精细的止损体系

# W2止损：W2低点下方3%（或前一波低点）
# W4止损：fib_618下方2%（最科学）
# C浪止损：C浪低点的103%（给3%容错空间）

# 增加：动态调整止损
def _calc_advanced_stop_loss(self, entry_price: float, wave_type: str):
    """
    根据波浪类型和入场位置动态计算止损
    """
    s = self.snapshot
    
    if wave_type == 'W2':
        # W2止损：W2低点下方
        base_stop = s.w2_end * 0.97
        # 同时检查是否跌破W1起点（趋势破坏）
        if s.w1_start:
            trend_stop = s.w1_start * 0.98
            return min(base_stop, trend_stop)  # 取较近的止损
    
    elif wave_type == 'W4':
        # W4止损：fib_618 或 W1高点（谁高谁作为止损）
        # W4跌破W1高点 = 趋势破坏，必须走
        if s.w1_end:
            trend_stop = s.w1_end * 0.98
            if s.fib_618:
                return max(s.fib_618, trend_stop)  # 取较远的（保守）
            return trend_stop
        return s.fib_618 if s.fib_618 else s.w4_end * 0.97
    
    elif wave_type == 'C':
        # C浪止损：C浪低点的103%
        return s.bi_low * 1.03
    
    return entry_price * 0.95  # 默认5%
```

#### 问题五：缺少大盘环境过滤

**问题**：当前只有周线方向过滤，缺少**大盘趋势**判断

**修正建议**：
```python
def get_signal(self) -> Dict:
    """
    增加大盘环境判断
    """
    signals = self.daily_cache.counter.get_buy_sell_signals()
    
    # 获取大盘状态（需要对接大盘数据）
    market_state = self._get_market_state()  # 'B'（资金流入）/ 'S'（资金流出）
    
    if market_state == 'S':
        sig = signals.get('signal', '')
        if sig in ('W2_BUY', 'W4_BUY'):
            signals = dict(signals)
            signals['status'] = 'INVALID'
            signals['invalidated_reason'] = '大盘资金流出，停止买入'
            signals['confidence'] = 0.0
            logger.info(f"[{self.symbol}] 大盘S点，拒绝{sig}信号")
    
    return signals
```

### 5.2 胜率影响因素（按重要性排序）

```
1. 大盘环境（最重要）
   ✅ 大盘资金翻红（B点）+ 买点 = 高胜率
   ❌ 大盘资金翻绿（S点）+ 任何买点 = 低胜率甚至空仓

2. 级别（次重要）
   ✅ 周线买点 > 日线买点 > 60分钟买点
   日线以下级别买点胜率急剧下降

3. 位置叠加（第三重要）
   ✅ 买点 + 均线支撑 + 斐波那契位 + 前高/低 = 高确定性
   ❌ 单独使用任何单一条件 = 中低胜率

4. 量价配合（第四重要）
   ✅ 上涨放量 + 回调缩量 = 健康趋势
   ❌ 上涨缩量 + 回调放量 = 趋势破坏信号
```
