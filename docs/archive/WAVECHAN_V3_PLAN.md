### Phase 1.2：买卖点信号生成
**目标**：在正确的时机发出正确的信号

| 信号 | 触发条件 | 动作 |
|------|---------|------|
| W2买信号 | W2回撤50%-61.8% + 底分型 | 买 |
| W4买信号（预警） | W4进行中 + 日线走出向上笔 | 观察/准备 |
| W4买信号（确认） | 周线底分型确认 + W4不破前低 | 买 |
| W5卖信号 | W5突破 + 顶分型 + 背离 | 卖 |
| C买信号（熊市） | C浪5浪完成 + 量能萎缩 | 买 |

### Phase 1.3：止损体系
- 买入后跌破前低 = 止损
- 买入后浮亏 > 5% = 主动止损
- 所有止损必须有明确价位

## 五、核心设计原则（老板教导 2026-03-27）

### 信号体系：假设 + 持续验证

```
不是：出现分型 = 信号确认
而是：假设这里终结 → 后续数据验证 → 确认/撤销
```

**三种信号状态：**
- `ALERT`：假设终结出现，需验证
- `CONFIRMED`：验证通过
- `INVALID`：验证失败

### 分型辅助判断规则

```
推动浪（与趋势同向）：
  W1起点 → 底分型    W1终点 → 顶分型
  W2起点 → 顶分型    W2终点 → 底分型
  W3起点 → 底分型    W3终点 → 顶分型
  W4起点 → 顶分型    W4终点 → 底分型
  W5起点 → 底分型    W5终点 → 顶分型

调整浪（与趋势反向）：
  A起点 → 顶分型    A终点 → 底分型
  B起点 → 底分型    B终点 → 顶分型
  C起点 → 顶分型    C终点 → 底分型
```

## 六、已确认的核心原理

### 波浪三大铁律
1. 浪2不创新低
2. 浪3不是最短
3. 浪4不重叠

### 买卖点原则
- 奇数浪(1,3,5)终结 = 卖点
- 偶数浪(2,4)终结 = 买点
- 牛市：A、B终结=卖，C终结=买
- 熊市：A、C终结=买，B终结=卖

### W4终结信号（老板教导）
- 周线出现底分型 ≈ W4 终结信号
- 日线走出向上笔 → W4 可能终结
- W4 进行中不猜终点

### 缠论+波浪结合原则
- 波浪定方向
- 缠论笔定精确入场
