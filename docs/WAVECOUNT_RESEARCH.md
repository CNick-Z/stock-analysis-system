## 🎯 核心框架（已确定）

```
周线确定趋势 → 日线寻找买点 → 斐波那契预判点位
```

### 第一步：CZSC笔替代Zigzag阈值
- ✅ CZSC笔 = 自然形成的走势分段，无需人为设定阈值
- ✅ 日线笔 ≈ 波浪中的细分浪
- ✅ 周线笔 ≈ 周线级别的波浪

### 第二步：波浪修正逻辑（老板核心教导）
**"1到5笔是一个完整的浪了，但是第6笔没有跌破第1笔的低点，这时候我们就要修正浪的识别了"**

**正确波浪识别流程：**
```
1. 第1个向下笔出现 → 记录为W1起点候选
2. 第2个向下笔出现 → 检查是否跌破W1起点
   - 跌破 → 重新扫描（W1起点下移）
   - 没跌破 → W2确认！之前的向上笔合并为W1
3. 新向上笔突破W1终点 → W3确认！
```

### 第三步：确认条件
**"因为第7笔创了新高，所以它不是反弹"**
```
向上笔突破前高 → 确认是驱动浪，不是反弹
向下笔跌破前低 → 确认是驱动浪，不是反弹
```

## 🔑 关键规则

### 波浪修正规则
1. **W2不能跌破W1起点** — 否则趋势破坏
2. **W3/W5是驱动浪** — 必须突破前一个驱动浪的高点
3. **W4不能跌破W3起点** — 否则趋势破坏
4. **等确认** — 不预判，走出来之后确认

### 多级别联动
- **周线笔** = 确定大趋势（W1/W2/W3）
- **日线笔** = 细分细浪
- **日线顶底分型** = 具体买卖点

### 斐波那契应用
```
W4回调位：0.382/0.500/0.618
W3目标：通常 = W1 × 1.618
```
